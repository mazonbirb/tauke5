Model notes

Dynamic fare model. Fares calculated from base + per-km rates (round-trip × 2):
• Yoxi Sedan: NT$120 base + NT$22/km  ·  Yoxi MPV: NT$150 base + NT$28/km
• iRent: NT$200 rental base + NT$15/km + NT$3.5/km fuel (round-trip)
Hotai variable cost = driver pay, fuel, depreciation (≈ 30% sedan, 27% MPV, 18% iRent rental).

c = Hotai's effective cost to issue 1 Hotai Point. Redemption liability NT$0.30–0.70 (varies by breakage rate) + platform overhead NT$0.05–0.15/pt (shrinks at scale).

Variable costs are NOT in c. They are already deducted from transport fare to get net Y. Adding them to c again would double-count.

Why Yoxi +50% vs iRent +20%

Yoxi carries a larger bonus because Hotai operates the full driver network — recovering fixed driver costs directly. iRent is a Hotai-owned self-service fleet where the host drives, so Hotai's marginal contribution is lower. The bonus gap incentivises hosts to prefer Yoxi.

Why the banquet model matters — supply-side logic

Self-sustaining supply. In Scenario A, the host earns points as a reward for effort (hosting, language, logistics). In Scenario B, the host's primary motivation is personal — they genuinely want to eat 辦桌. ChicTrip converts that existing desire into a social event that generates matching fees and Yoxi rides for Hotai.

No subsidy needed. Unlike platform labour models where supply must be incentivised with cash, 辦桌 supply is driven by hunger. Points are a bonus, not the reason. This structurally lowers host acquisition cost to near zero.
